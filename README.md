# Franklin · Shawarma, Burgers & More

A restaurant website for **Franklin** — *"It's not just Food, It's an Experience."*

## 🔗 Live Site

**https://my-app-tau-gules-82.vercel.app**

## Tech Stack

- [Next.js 16](https://nextjs.org/) (App Router, Turbopack)
- React + TypeScript
- Tailwind CSS
- Deployed on [Vercel](https://vercel.com/)

## Project Structure

The Next.js application lives in the [`my-app`](./my-app) directory.

```
franklin/
└── my-app/        # Next.js app (pages, components, data)
```

## Local Development

```bash
cd my-app
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Deployment

Deployed to Vercel from the `my-app` directory:

```bash
cd my-app
vercel --prod
```
